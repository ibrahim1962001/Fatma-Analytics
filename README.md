## Local Development

1. **Clone the repository**
   ```bash
   git clone https://github.com/ibrahim1962001/Fatma-Analytics.git
   cd Fatma-Analytics
   ```

2. **Open locally** — any static file server works, or open `index.html` directly in a browser.

   ```bash
   # Option A: Python
   python -m http.server 8080

   # Option B: Node (if npx is available)
   npx serve .
   ```

3. Visit `http://localhost:8080` (or your chosen port).

## Deploy to GitHub Pages

This repo includes [`.github/workflows/pages.yml`](.github/workflows/pages.yml) for automatic deployment on every push to `main`.

**Manual setup (one time):**

1. Go to **Settings → Pages**
2. **Build and deployment → Source:** GitHub Actions
3. Push to `main` — the workflow publishes the site automatically

**Expected URL:** `https://ibrahim1962001.github.io/Fatma-Analytics/`
